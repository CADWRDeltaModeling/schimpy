Installation¶

Stable release¶

We recommend conda for installing schimpy. The simplest way to install schimpy is to run this command in your terminal:

$ conda install -c cadwr-dms schimpy

For greater flexibility and a set of tools that are commonly used with SCHISM, however, you may want to install a more complete schism environment using the schism_environment.yml file that is in the parent directory of BayDeltaSCHISM

From sources¶

The sources for schimpy can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone https://github.com/CADWRDeltaModeling/schimpy

Or download the tarball:

$ curl -OJL https://github.com/CADWRDeltaModeling/schimpy/tarball/master

Once you have a copy of the source, you can install it with:

# make sure all dependencies from setup_devenv.bat are installed by conda
$ setup_devenv.bat # or copy and paste the commands into shell
$ conda activate dev_schimpy
$ pip install --no-deps -e .
